## ğŸ“Š Calculations

### Subtotal
```
Subtotal = SUM(item.total for each item)
         = SUM(quantity Ã— sale_price/price)
```

### Savings
```
Savings = SUM((price - sale_price) Ã— quantity)
        = Original Total - Subtotal
```

### Tax (10%)
```
Tax = Subtotal Ã— 0.10
```

### Grand Total
```
Grand Total = Subtotal + Tax + Shipping
            = Subtotal Ã— 1.10  (since Shipping is FREE)
```
